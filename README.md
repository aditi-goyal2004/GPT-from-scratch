# GPT from Scratch

A simple implementation of a GPT-style language model trained on the **Tiny Shakespeare** dataset using PyTorch.

---

## Features

- Character-level GPT model
- Multi-head self-attention
- Generate Shakespeare-like text
- Trained entirely from scratch

---

## Requirements

- Python 3.8+
- PyTorch
- NumPy

Install dependencies with:

```bash
pip install torch torchvision numpy
